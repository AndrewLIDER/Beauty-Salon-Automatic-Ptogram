# Beauty Pro - Система автоматизації салону краси

Повнофункціональна система для управління салоном краси з онлайн бронюванням, управлінням клієнтами та адмін-панеллю.

## Основні функції

- **Онлайн бронювання 24/7** - Клієнти можуть записатися на прийом в будь-який час
- **Управління послугами** - Повний каталог послуг з цінами та тривалістю
- **Управління майстрами** - Інформація про майстрів, їх спеціалізації та графіки роботи
- **Адмін-панель** - Управління записами, зміна статусів, перегляд статистики
- **Система автентифікації** - Безпечний вхід для клієнтів та адміністраторів
- **Адаптивний дизайн** - Працює на всіх пристроях

## Налаштування

### 1. Налаштування Supabase

У файлі `.env.example` вказано необхідні змінні оточення. Створіть файл `.env` та додайте ваші дані Supabase:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

### 2. База даних

База даних вже налаштована з наступними таблицами:
- `profiles` - Профілі користувачів
- `services` - Послуги салону
- `masters` - Майстри
- `bookings` - Записи клієнтів
- `working_hours` - Графік роботи майстрів
- `master_services` - Зв'язок між майстрами та послугами

Тестові дані вже додані:
- 8 послуг (манікюр, педикюр, нарощування)
- 3 майстри
- Графіки роботи

### 3. Створення адміністратора

Для доступу до адмін-панелі потрібно створити користувача з роллю `admin`:

1. Зареєструйтеся через інтерфейс
2. В базі даних Supabase знайдіть вашого користувача в таблиці `profiles`
3. Змініть поле `role` на `admin`

## Використання

### Для клієнтів:
1. Відкрийте головну сторінку
2. Перегляньте послуги та майстрів
3. Заповніть форму бронювання
4. Очікуйте підтвердження

### Для адміністраторів:
1. Увійдіть з обліковим записом адміністратора
2. Переглядайте всі записи в адмін-панелі
3. Підтверджуйте, завершуйте або скасовуйте записи
4. Переглядайте статистику

## Технології

- **React 18** - UI фреймворк
- **TypeScript** - Типізація
- **Vite** - Збірка проекту
- **Tailwind CSS** - Стилізація
- **Supabase** - Backend та база даних
- **Lucide React** - Іконки

## Дизайн

Система використовує голубу кольорову схему (cyan/blue) з фотографіями робочих місць манікюру та педикюру з Pexels. Дизайн сучасний, чистий та професійний.
